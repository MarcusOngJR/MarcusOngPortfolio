Overview

SeniorSync is an MVP designed to improve quality-of-life for elderly users and caretakers. The project was developed using Scrum practices, where I contributed as both Scrum Master and developer. This project was initially done without a live database integration but I have set up a database with a server hosting this project for the live demo.
